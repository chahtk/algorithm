## 문제
두 개의 정수 입력 a, b를 받아서 a+b를 출력하시오.

### 입력
a b; a와 b는 -100,000과 100,000 사이의 정수이다.

### 출력
a+b

<br>

## 풀이

### method1
- input 사용
- 입력값을 각각 형변환

### method2
- 입력값을 map 함수를 통해 바로 형변환
- map은 리스트의 요소를 지정된 함수로 처리해주는 함수(map은 원본 리스트를 변경하지 않고 새 리스트를 생성)

### method3
- sys.stdin.readline: 한 줄을 읽어드린다. input() 보다 빠름
